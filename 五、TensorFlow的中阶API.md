# 五、TensorFlow的中阶API

TensorFlow的中阶API主要包括: 

* 数据管道(tf.data)

* 特征列(tf.feature_column)

* 激活函数(tf.nn)

* 模型层(tf.keras.layers)

* 损失函数(tf.keras.losses)

* 评估函数(tf.keras.metrics)

* 优化器(tf.keras.optimizers)

* 回调函数(tf.keras.callbacks)

如果把模型比作一个房子，那么中阶API就是【模型之墙】。
